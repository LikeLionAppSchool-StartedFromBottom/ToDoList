## ☄️Trouble Shooting
 - 리스트 배열에 엔터를 누를 때 onCommit을 사용 시 clear가 되지 않음
	 - save 버튼으로 구현한 경우 function 내의 clear 기능이 작동함
- 수정, 삭제 버튼을 HStack으로 넣는 경우 HStack의 모든 버튼이 실행
